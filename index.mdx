```bash
curl -X POST https://ninjachat.ai/api/v1/chat \
  -H "Authorization: Bearer nj_sk_YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "gpt-5", "messages": [{"role": "user", "content": "Hello!"}]}'
```

One API key gives you GPT-5, Claude, Gemini, FLUX, Imagen, Veo, and 25+ more models. Same endpoint, same format — just change the `model` parameter.

## How it works

Every endpoint follows the same pattern:

```
POST https://ninjachat.ai/api/v1/{endpoint}
Authorization: Bearer nj_sk_YOUR_API_KEY
Content-Type: application/json

{"model": "model-id", ...params}
```

**Chat and images** return results immediately. **Video** is async — you submit a job and [poll for the result](/video-generation).

<Tip>
**Want to try before coding?** The [API Playground](https://www.ninjachat.ai/api-playground) lets you test all models in your browser — no code required.
</Tip>

Every response includes a `cost` field so you always know exactly what you paid:

```json
{
  "message": {"role": "assistant", "content": "..."},
  "cost": {"this_request": "$0.006"}
}
```
