The NinjaChat API gives you access to AI chat completions, web search, and image generation. All models are uncensored with no content filters. All endpoints require a paid NinjaChat subscription and an API key.

<CardGroup cols={3}>
  <Card title="Chat Completions" icon="message" href="/api-reference/chat">

  </Card>
  <Card title="Web Search" icon="magnifying-glass" href="/api-reference/search">
    Search the web, academic papers, or news with AI-synthesized answers.
  </Card>
  <Card title="Image Generation" icon="image" href="/api-reference/images">
    Generate images from text prompts, including image-to-image editing.
  </Card>
</CardGroup>

## Base URL

All API requests use the following base URL:

```
https://ninjachat.ai
```
