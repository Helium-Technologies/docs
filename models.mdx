## Switching between models

All models within a type share the same endpoint and request format. Switch by changing `model`:

```python
import requests

headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer nj_sk_YOUR_API_KEY",
}

# Use GPT-5
requests.post("https://ninjachat.ai/api/v1/chat", headers=headers, json={
    "model": "gpt-5",
    "messages": [{"role": "user", "content": "Hello!"}]
})

# Switch to Claude — same code, different model
requests.post("https://ninjachat.ai/api/v1/chat", headers=headers, json={
    "model": "claude-sonnet-4.6",
    "messages": [{"role": "user", "content": "Hello!"}]
})
```

<Note>
  The `model` parameter is optional for chat and images. If omitted, the API uses a default model. We recommend always specifying the model explicitly.
</Note>
