## Chat Models — `POST /api/v1/chat`

```bash
  -H "Authorization: Bearer nj_sk_YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "MODEL_ID", "messages": [{"role": "user", "content": "Hello"}]}'
```

## Video Models — `POST /api/v1/video`

All video models cost **$3.00/video**. Video is async — [see the guide](/video-generation) for polling code.

```bash
curl -X POST https://ninjachat.ai/api/v1/video \
  -H "Authorization: Bearer nj_sk_YOUR_API_KEY" \
```
