<Tip>
**Which video model?** Start with `veo-3.1-fast` ($3.00, good balance). Use `runway-gen4.5` ($5.00) or `veo-3.1` ($5.00) for best quality. Use `seedance-2` ($3.00) for longest clips (up to 15s). [Compare all →](/models)

Video generation is **async**: submit a job → poll for status → get an MP4 URL.

## Complete example (copy-paste-run)

<CodeGroup>

```python Python
import os, time, requests

API = "https://ninjachat.ai"
HEADERS = {"Authorization": f"Bearer {os.environ['NINJACHAT_API_KEY']}"}

# 1. Submit
r = requests.post(f"{API}/api/v1/video", headers=HEADERS, json={
    "model": "google-veo-2",
    "prompt": "A timelapse of a flower blooming, macro lens, soft lighting",
    "duration": 5,
    "aspect_ratio": "16:9"
})
r.raise_for_status()
job = r.json()
request_id = job["request_id"]
print(f"Submitted: {request_id} (cost: {job['cost']['this_request']})")

# 2. Poll every 10 seconds
while True:
    s = requests.get(f"{API}/api/v1/video/status",
        headers=HEADERS, params={"request_id": request_id}).json()

    if s["status"] == "completed":
        print(f"Done! {s['result']['video_url']}")
        break
    if s["status"] == "failed":
        print(f"Failed: {s.get('error')} (you were NOT charged)")
        break

    print(f"  {s.get('progress', 0)}%...")
    time.sleep(10)
```

```javascript Node.js
const API = "https://ninjachat.ai";
const HEADERS = {
  "Authorization": `Bearer ${process.env.NINJACHAT_API_KEY}`,
  "Content-Type": "application/json",
};

const r = await fetch(`${API}/api/v1/video`, {
  method: "POST", headers: HEADERS,
  body: JSON.stringify({
    model: "google-veo-2",
    prompt: "A timelapse of a flower blooming, macro lens, soft lighting",
    duration: 5, aspect_ratio: "16:9",
  }),
});
const job = await r.json();
console.log(`Submitted: ${job.request_id} (cost: ${job.cost.this_request})`);

// 2. Poll every 10 seconds
while (true) {
  const s = await fetch(
    `${API}/api/v1/video/status?request_id=${encodeURIComponent(job.request_id)}`,
    { headers: HEADERS }
  ).then(r => r.json());

  if (s.status === "completed") { console.log(`Done! ${s.result.video_url}`); break; }
  if (s.status === "failed") { console.log(`Failed: ${s.error} (not charged)`); break; }
  console.log(`  ${s.progress || 0}%...`);
  await new Promise(r => setTimeout(r, 10000));
}
```

## Submit request

```
POST https://ninjachat.ai/api/v1/video
```

| Parameter | Type | Required | Default | Description |
|---|---|---|---|---|
| `prompt` | string | **Yes** | — | Describe the video. Include camera movement, action, style. Max 4,000 chars. |
| `model` | string | No | `seedance-2` | Model ID. See table below. |
| `duration` | integer | No | 8 | Length in seconds. Range depends on model. |
| `aspect_ratio` | string | No | `16:9` | `16:9` or `9:16`. Ignored by Seedance 2. |
| `image_url` | string | No | — | Animate this image. Must be public HTTPS URL. |

### Submit response

```json
{
  "request_id": "gemini:abc123def456",
  "model": "google-veo-2",
  "cost": { "this_request": "$3.00" },
  "balance": "$37.00"
}
```

## Poll for status

```
GET https://ninjachat.ai/api/v1/video/status?request_id={request_id}
```

Polling is **free** — no credits, no rate limit.

### Status: processing

```json
{ "status": "processing", "progress": 45 }
```

### Status: completed

```json
{
  "status": "completed",
  "result": { "video_url": "https://cdn.ninjachat.ai/videos/abc123.mp4" }
}
```

### Status: failed

```json
{ "status": "failed", "error": "Video generation failed" }
```

Failed jobs are **not charged** — credits refunded automatically.

## Models

|---|---|---|---|---|
| Seedance 2 | `seedance-2` | 5–15s | ~1-2 min | Unrestricted content, longest clips |
| Kling Video | `kling-video` | 5–10s | ~3-4 min | Motion quality, coherence |
| Google Veo 2 | `google-veo-2` | 5–8s | ~40s | $3.00 — Fastest + high quality |
| Veo 3.1 | `veo-3.1` | 4, 6, 8s | ~3-5 min | $5.00 — Best quality |
| Veo 3.1 Fast | `veo-3.1-fast` | 4, 6, 8s | ~1-2 min | $3.00 — Speed-optimized |
| Veo 3 Fast | `google-veo-3-fast` | 4, 6, 8s | ~1-2 min | $3.00 — Budget Veo |
| Runway Gen-4.5 | `runway-gen4.5` | 5 or 10s | ~1-3 min | $5.00 — Runway flagship, cinematic |
| Runway Gen-4 Turbo | `runway-gen4-turbo` | 5 or 10s | ~30-60s | $3.00 — Fast image-to-video (requires image_url) |

## Image-to-video

Animate a still image:

```python
r = requests.post(f"{API}/api/v1/video", headers=HEADERS, json={
    "model": "veo-3.1",
    "prompt": "Camera slowly pulls back, clouds drift across the sky",
    "duration": 6,
    "image_url": "https://example.com/landscape.jpg"
})
# Then poll as shown above
```

## Prompt tips

| Weak | Strong |
|---|---|
| `mountains` | `Drone shot slowly ascending over a misty mountain valley at golden hour, camera reveals a river below, cinematic` |
| `ocean` | `Underwater camera glides through a coral reef, bioluminescent creatures pulse with light, slow motion, nature documentary` |

Include: **camera movement** + **subject action** + **style** + **lighting**
