## Prerequisites

- An active API subscription ($5/month)
- Prepaid credits in your account
- An API key from your [dashboard](https://www.ninjachat.ai/settings?tab=api)

## 1. Subscribe and add credits

Sign in to [ninjachat.ai](https://www.ninjachat.ai) and go to [Settings > API](https://www.ninjachat.ai/settings?tab=api) to subscribe ($5/month) and add credits (minimum $10).

## 2. Create an API key

From the same [settings page](https://www.ninjachat.ai/settings?tab=api), create a new API key. Your key starts with `nj_sk_` and is shown only once — store it securely.

## 3. Make your first request

<CodeGroup>

```bash cURL
curl -X POST https://ninjachat.ai/api/v1/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer nj_sk_YOUR_API_KEY" \
  -d '{"messages": [{"role": "user", "content": "Hello!"}]}'
```

```python Python
import requests

response = requests.post(
    "https://ninjachat.ai/api/v1/chat",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer nj_sk_YOUR_API_KEY",
    },
    json={
        "messages": [
            {"role": "user", "content": "Hello!"}
        ]
    },
)

print(response.json())
```

```javascript JavaScript
const response = await fetch("https://ninjachat.ai/api/v1/chat", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer nj_sk_YOUR_API_KEY",
  },
  body: JSON.stringify({
    messages: [
      { role: "user", content: "Hello!" }
    ],
  }),
});

const data = await response.json();
console.log(data.message.content);
```

</CodeGroup>
