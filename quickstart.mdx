## 1. Get an API key

Go to your [Developers dashboard](https://www.ninjachat.ai/developers/keys):
1. Add credits to your account
2. Click **Create API Key**, give it a name, and copy it immediately (starts with `nj_sk_`, shown once)

No subscription required. Pay only for what you use.

## 2. Make a request

<CodeGroup>

```bash cURL
curl -X POST https://ninjachat.ai/api/v1/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer nj_sk_YOUR_API_KEY" \
  -d '{
    "model": "gpt-5",
    "messages": [{"role": "user", "content": "What is the capital of France?"}]
  }'
```

```python Python (no dependencies except requests)
import requests

r = requests.post("https://ninjachat.ai/api/v1/chat",
    headers={"Authorization": "Bearer nj_sk_YOUR_API_KEY"},
    json={
        "model": "gpt-5",
        "messages": [{"role": "user", "content": "What is the capital of France?"}]
    }
)
print(r.json()["message"]["content"])
```

```javascript Node.js (zero dependencies)
const r = await fetch("https://ninjachat.ai/api/v1/chat", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer nj_sk_YOUR_API_KEY",
  },
  body: JSON.stringify({
    model: "gpt-5",
    messages: [{ role: "user", content: "What is the capital of France?" }],
  }),
});
const data = await r.json();
console.log(data.message.content);
```

</CodeGroup>

```python Video
# Step 1: Submit
r = requests.post("https://ninjachat.ai/api/v1/video",
    headers={"Authorization": "Bearer nj_sk_YOUR_API_KEY"},
    json={
        "model": "google-veo-2",
        "prompt": "A timelapse of a flower blooming",
        "duration": 5
    }
)
request_id = r.json()["request_id"]

# Step 2: Poll every 10s
import time
while True:
    s = requests.get(
        f"https://ninjachat.ai/api/v1/video/status?request_id={request_id}",
        headers={"Authorization": "Bearer nj_sk_YOUR_API_KEY"}
    ).json()
    if s["status"] == "completed":
        print(s["result"]["video_url"])
        break
    if s["status"] == "failed":
        print("Failed:", s.get("error"))
        break
    time.sleep(10)
```

